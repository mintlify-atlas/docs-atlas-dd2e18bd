Field Theory is a local-first CLI that downloads your X/Twitter bookmarks to your machine, builds a full-text search index, classifies them by topic, and makes them available to any AI agent with shell access. Your data never leaves your machine.

## Get started in 60 seconds

<Steps>
  <Step title="Install Field Theory">
    ```bash
    npm install -g fieldtheory
    ```
    Requires Node.js 20+ and Google Chrome.
  </Step>
  <Step title="Open Chrome and log into X">
    Field Theory reads your Chrome session to authenticate — no passwords or API keys needed.
  </Step>
  <Step title="Sync your bookmarks">
    ```bash
    ft sync
    ```
    Your bookmarks are downloaded and indexed in `~/.ft-bookmarks/`.
  </Step>
  <Step title="Search and explore">
    ```bash
    ft search "distributed systems"
    ft viz
    ft categories
    ```
  </Step>
</Steps>

<Note>
